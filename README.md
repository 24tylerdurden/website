# Personal Website (GitHub Pages)

This is a ready-to-deploy personal website for GitHub Pages.

## Files
- `index.html`
- `styles.css`
- `script.js`

## Customize content
Edit these placeholders in `index.html`:
- `pavanIlla`
- `you@example.com`
- `https://github.com/pavanIlla`
- `https://www.linkedin.com/in/yourusername`
- project titles/descriptions/links

## Host on GitHub Pages

### Option A: User site (`username.github.io`)
1. Create a GitHub repository named exactly `username.github.io`.
2. Push this code to the `main` branch.
3. GitHub Pages will serve automatically from `main` branch root.

### Option B: Project site (`username.github.io/repo-name`)
1. Create any repository name.
2. Push this code.
3. In GitHub: `Settings` -> `Pages`.
4. Under `Build and deployment`, set:
   - Source: `Deploy from a branch`
   - Branch: `main` and folder `/ (root)`

After deploy, your site URL appears in the Pages settings.

## Local preview
Open `index.html` in a browser.
